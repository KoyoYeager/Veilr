### 3.4 全画面モード操作フロー

```
[🖥全画面] クリック
    │
    ├── モニター1台 → 即座にそのモニター全画面化
    │
    └── マルチモニター → サブメニュー表示
                         ├── モニター1 (1920×1080)
                         ├── モニター2 (2560×1440)
                         └── ...
                              │ 選択
                              ▼
                         選択モニター全画面化
                         （元のサイズ・位置を記憶）
                              │
                         [🖥解除] or ESC
                              │
                              ▼
                         元のサイズ・位置に復帰
```

### 3.5 ツールバーボタン

| ボタン | 表示テキスト | 動作 |
|--------|------------|------|
| モード切替 | シートモード中：「消去モードに切替」/ 消去中：「シートモードに切替」 | モードをトグル |
| 🖥 全画面 | 通常時：「全画面」/ 全画面中：「解除」 | モニター選択→全画面化 or 解除 |
| 📷 保存 | 「保存」 | シート範囲の色抜き画像を保存（W07ダイアログ表示） |
| ⚙ 設定 | ― | W03設定ウィンドウを表示 |
| ✕ 閉じる | ― | ウィンドウを非表示（タスクトレイに格納） |

### 3.6 左上モード表示

| モード | 表示テキスト |
|--------|------------|
| シートモード | `シートモード` |
| 消去モード | `消去モード` |
| 全画面時 | `シートモード（全画面）` / `消去モード（全画面）` |

- ドラッグバー左端に半透明テキストで表示
- フォント：Segoe UI 11px、半透明（他のUI要素と同じ透明度）

### 3.8 共通仕様

| 項目 | 内容 |
|------|------|
| ウィンドウスタイル | `WindowStyle=None` / `AllowsTransparency=True` |
| 最前面固定 | `Topmost=True` + `HWND_TOPMOST` |
| ドラッグ移動 | ドラッグバー（上端）または本体ドラッグで移動 |
| リサイズ | 角4箇所＋辺4箇所にリサイズハンドル |
| 初期サイズ | 300×200px（前回値があれば復元） |
| 複数配置 | 同時に複数シートを配置可能 |
| **全要素半透明** | ドラッグバー・ツールバー・ボタン文字すべて半透明で描画 |

### 3.9 半透明表示ルール

| 要素 | 透明度 |
|------|--------|
| シート本体（シートモード） | 40〜80%（設定で調整、デフォルト50%） |
| ドラッグバー | シート本体より10%濃く |
| ツールバー背景 | シート本体より15%濃く |
| ボタン文字・アイコン | シート本体と同じ透明度 |
| ホバー時のボタン | 不透明度+20%（視認性向上） |

### 3.10 モード切替動作

```
シートモード（デフォルト）
    │ 左上: シートモード
    │ ボタン: [消去モードに切替]
    │
    │ クリック
    ▼
消去モード
    │ 左上: 消去モード
    │ ボタン: [シートモードに切替]
    │ シート範囲のスクリーンキャプチャ開始
    │ ターゲット色を周囲色で塗りつぶし
    │ 100〜500ms間隔で更新
    │
    │ クリック
    ▼
シートモードに戻る
```

### 3.11 表示/非表示

| 操作 | 動作 |
|------|------|
| `Ctrl+Shift+E` | 表示/非表示トグル |
| トレイ左クリック | 表示/非表示トグル |
| ✕ボタン | 非表示（タスクトレイに格納） |
| 非表示→表示 | 前回の位置・サイズ・モードを復元 |

## 6. W05: 閾値プレビュー

### 6.1 レイアウト

```
┌─────────────────────────────────────────────────┐
│ ┊                                               ┊│
│ ┊  対象ピクセルを点滅ハイライト表示              ┊│
│ ┊                                               ┊│
│ ┊  ██ ← 閾値に該当するピクセル                  ┊│
│ ┊     （アクセントカラーで点滅 500ms間隔）        ┊│
│ ┊                                               ┊│
│ ┊  他のピクセルは半透明で薄く表示                 ┊│
│ ┊                                               ┊│
│ └───────────────────────────────────────────────┘│
│                                                  │
│  ┌──────────────────────────────────────────────┐│
│  │ 対象: 1,234 px / 全体: 480,000 px (0.26%)   ││ ← 情報バー
│  │                     [ ✓ 適用 ]  [ ✕ 戻る ]  ││
│  └──────────────────────────────────────────────┘│
└──────────────────────────────────────────────────┘
```

### 6.2 仕様

| 項目 | 内容 |
|------|------|
| 表示トリガー | 設定画面の [ ▶ プレビュー ] ボタン |
| ハイライト | 対象ピクセルをアクセントカラーで500ms点滅 |
| 非対象 | 元画像を半透明（opacity 30%）で表示 |
| 情報バー | 対象ピクセル数 / 全体ピクセル数 / 割合を表示 |
| リアルタイム連動 | 設定画面のスライダー変更に即座に追従 |
| 適用 | 現在の閾値で確定し、処理オーバーレイに戻る |
| 戻る | 変更を破棄して設定画面に戻る |

### 9.2 WPF ResourceDictionary構成

```
Resources/
├── Themes/
│   ├── DarkTheme.xaml
│   └── LightTheme.xaml
├── Styles/
│   ├── ButtonStyles.xaml
│   ├── SliderStyles.xaml
│   ├── TabControlStyles.xaml
│   └── TextBoxStyles.xaml
└── AccentColors/
    └── AccentBrushes.xaml   ← 動的に差し替え
```

テーマ切替時は `Application.Current.Resources.MergedDictionaries` を差し替える。

### 9.3 多言語対応（i18n）

- WPF標準の `.resx` リソースファイルで管理
- `Strings.ja.resx`（日本語）/ `Strings.en.resx`（英語）
- 設定変更時に `CultureInfo` を切り替え、UIに即座に反映
- OS言語を自動検出し、初回起動時のデフォルト言語を決定

## 2. W02: シートウィンドウ

### 2.1 レイアウト

```
┌──────────────────────────────────────────┐ ← 設定色のボーダー(1px)
│ シートモード [GPU]                    ✕  │ ← ドラッグバー(24px)
│                                          │
│          処理済み画像表示エリア            │ ← Image (Stretch=Fill)
│          (WriteableBitmap)               │
│                                          │
│ [消去モードに切替] [全画面] [保存] [設定] │ ← ツールバー(32px)
└──────────────────────────────────────────┘
```

### 2.2 仕様

| 項目 | 内容 |
|------|------|
| WindowStyle | None（カスタム枠） |
| AllowsTransparency | True |
| Topmost | True（常に最前面） |
| ShowInTaskbar | False |
| ResizeMode | CanResize（全辺・全角からリサイズ、WM_NCHITTEST実装） |
| ボーダー | 1px、設定色 |
| 角丸 | CornerRadius=8 |

### 2.3 ドラッグバー（上部24px）

| 項目 | 内容 |
|------|------|
| 左側テキスト | モードラベル + 処理モード（例: `シートモード [GPU]`） |
| 右側 | ✕ボタン（非表示→トレイ格納） |
| ドラッグ | MouseDown/Move/Up（OnRenderingに同期） |
| 背景 | シートモード: 半透明シート色 / 消去モード: 半透明白 |
| 文字色 | シートモード: 自動コントラスト / 消去モード: 黒 |

### 2.4 ツールバー（下部32px）

| ボタン | 動作 |
|--------|------|
| 消去モードに切替 / シートモードに切替 | モード切替 + キャプチャ |
| 全画面 / 解除 | 全画面トグル |
| 保存 | シート範囲の色抜き画像をエクスポート |
| 設定 | 設定ウィンドウを開く |

### 2.5 バー透明度

| モード | バー背景 | 文字色 |
|--------|---------|--------|
| シートモード | 半透明シート色（設定値%） | 自動コントラスト（白/黒） |
| 消去モード | 半透明白（設定値%） | 黒 |

透明度は設定「バー透明度」スライダーで0-100%調整可能（デフォルト70%）。

## 3. W03: 設定ウィンドウ

### 3.1 タブ構成

| タブ | 内容 |
|------|------|
| 色設定 | シート色、透明度、消去アルゴリズム、許容範囲 |
| 動作 | 自動更新、更新頻度、バー透明度、GPU高速化 |
| ホットキー | シート表示/非表示キー |
| 外観 | 言語（日本語/英語） |

### 3.2 色設定タブ

- カラープレビュー + HEX入力
- 変更ボタン（ColorDialog）
- スポイトボタン（全画面スナップショット方式）
- シート透明度スライダー（10-100%）
- 消去アルゴリズム: Chroma Key / Lab Mask / YCbCr（ラジオボタン + 説明文）
- 許容範囲スライダー（0-100、厳密↔柔軟ラベル）

### 3.3 動作タブ

- 自動更新チェックボックス
- 更新頻度スライダー（1ms MAX〜500ms 2fps、fps表示付き）
- バー透明度スライダー（0-100%）
- GPU高速化チェックボックス

### 3.4 ボタン

- 適用 → 設定保存 + アプリ再起動
- キャンセル → 閉じる
- OK → 適用 + 閉じる

## 4. タスクトレイ

### 4.1 メニュー

```
├── シートを表示 / 非表示
├── ───────
├── 設定...
├── ───────
└── 終了
```

### 4.2 操作

| 操作 | 動作 |
|------|------|
| 左クリック | シート表示/非表示トグル |
| ダブルクリック | 設定ウィンドウを開く |
| 右クリック | コンテキストメニュー |

### 4.3 ホットキー

| キー | 動作 |
|------|------|
| Ctrl+Shift+E | シート表示/非表示（デフォルト） |
| F5 / Space | 手動リフレッシュ（シートウィンドウ上） |
| Escape | 全画面解除 |
